# Support

For questions, bugs, or feature requests, use GitHub Issues:

```text
https://github.com/xiaogehenjimo/codebuddy-hud/issues
```

When reporting a rendering issue, include:

- Node.js version
- CodeBuddy Code version
- Terminal name
- `config.json` with sensitive paths removed
- Output from `codebuddy-hud inspect` with private values removed
